# 🎬 AI YouTube Video Question Answering System

An AI-powered web application that allows users to ask questions about any YouTube video. The system automatically fetches the video transcript, processes it using LangChain and large language models, and provides accurate answers based only on the video’s content.

---

## 🚀 Features

- Ask questions from any YouTube video
- Uses transcript + AI (LangChain + LLM)
- Supports multilingual videos
- Semantic search using FAISS vector database
- Clean and simple Streamlit interface
- Answers strictly based on video content

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- OpenAI / GitHub Models API
- YouTube Transcript API
